# Proxy Scraper & Cleaner

> *İnternetten proxy toplama, çalma, temizleme ve düzenleme aracı*

## 🎭 Proje Hakkında

Bu tool, internet üzerindeki proxy kaynaklarından HTTP, HTTPS ve SOCKS5 proxy'leri toplar, veya girilen site urlesinden o sitenin proxylerini çalar, test eder ve kullanıma hazır hale getirir.

---

## 🚀 Özellikler

| Özellik | Açıklama |
|---------|-----------|
| **Proxy Toplama** | 6+ farklı kaynaktan proxy çeker |
| **Proxy Temizleme** | Çalışmayan proxy'leri tespit edip temizler |
| **Proxy Düzenleme** | Tiplerine göre proxy'leri ayıklar ve birleştirir |
| **Web'den Çalma** | Herhangi bir web sitesinden IP:Port formatında proxy'leri çeker |
| **Proxy Test** | Gerçek hedeflere istek atarak proxy'leri test eder |
| **SOCKS5 Desteği** | SOCKS5 proxy'lerini de destekler |
